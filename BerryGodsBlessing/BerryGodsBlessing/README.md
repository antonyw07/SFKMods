# Berry God's Blessing

A lightweight BepInEx mod for **Super Fantasy Kingdom** that adds a feature granting blessings from the Berry God. The more berries bushes in your kingdom, the more likely you will be blessed by the Berry God. The Berry God's blessings have various effects that can benefit the kingdom.
